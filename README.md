Repository of the MobCapture plugin.

## Commands

For `/mc` you can alternatively use `/mobcapture`

- `/mc` - Shows the help page.
- `/mc help` - Shows the help page.
- `/sc reload` - Reloads the config file.

## Permissions

- `mobcapture.*` - Access to all features.
